# 修改文件分类

## 特性描述
用户可以修改一个文件的分类
优先级=高

## 刺激响应序列
刺激：用户请求改变某个文件的类别

响应：系统提示用户选择新的文件类别

刺激：用户选择新的分类并确认

响应：系统更新文件类别并结束修改文件类别处理

## 相关功能需求

| 编号 | 需求描述 |
| --- | --- |
| File.ChangeClass | 系统允许用户修改文件类别 |
| File.ChangeClass.selectFile | 在修改类别时,系统应该请求用户选择文件 |
| File.ChangeClass.selectClass | 在修改类别时,系统应该请求用户选择一个已存在的类别 |
| File.ChangeClass.cancel | 用户可以取消修改类别 |
